# sales-forecasting

This project forecasts future sales using historical sales data.

## Objective
To help businesses plan inventory and revenue using predictive analytics.

## Methodology
- Trend and seasonality analysis
- Time series modeling (ARIMA, Regression)
- Model evaluation using RMSE and MAE

## Insights
The model captures demand patterns and provides accurate short-term sales forecasts.

## Tools
Python, Pandas, NumPy, Statsmodels, Matplotlib
